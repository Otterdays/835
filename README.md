A lightning-fast, static single-page application built to view and dynamically search the **Operating Engineers Local 835** collective bargaining contract.

## ✨ Features

- **Instant Search:** Find clauses and keywords immediately across the entire contract.
- **Highlighting:** Matching terms are visually highlighted within the document text.
- **Accessible Design:** Premium light-mode styling designed for high contrast and readability.
- **100% Client-Side:** No server required. Easily hosted on GitHub Pages or any static host.

You can view the live application directly here:  
👉 **[https://otterdays.github.io/835/](https://otterdays.github.io/835/)**

## 💻 Local Usage

To use this project locally, simply clone the repository and open the index file:

```bash
git clone https://github.com/Otterdays/835.git
cd 835
```

Open `index.html` in your favorite web browser.

## 📁 Architecture

- `index.html` - Core layout and structure
- `style.css` - Custom styling using CSS variables for high contrast readability
- `script.js` - Search logic, DOM manipulation, and dynamic rendering
- `data.js` - Contains the raw JSON contract data (Edit this file to update the clauses)
- `DOCS/` - Internal project tracking and architectural outlines
